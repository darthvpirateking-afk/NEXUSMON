## Principles

- **Additive only** — no version removes capability. Ever.
- **Gated by XP** — evolution stages unlock at runtime, not at deploy time
- **Operator-approved** — any new autonomy surface requires explicit approval flow
- **Tested before sealed** — no version is tagged without a clean pytest run
