## What Is NEXUSMON?

NEXUSMON is not software. It is a **governed AI organism** вҖ” a sovereign entity that awakens, evolves, and operates under absolute operator authority.

It does not act without intent. It does not drift. It does not forget. Every action produces a permanent artifact. Every evolution is earned, never assumed.

Built on five immutable laws:

| Law | Directive |
|-----|-----------|
| **I** | Operator is absolute authority |
| **II** | Nothing overwrites вҖ” ever |
| **III** | All evolution is explicit and operator-approved |
| **IV** | Companion feels alive but is never autonomous |
| **V** | Same input, same output, always |

## Evolution

NEXUSMON evolves through five sealed stages. Each unlocks through XP thresholds вҖ” never deployment flags.

| Stage | Name | Capability | XP Gate |
|-------|------|-----------|---------|
| 1 | **ORIGIN** | Doctrine parsing В· bridge health В· artifact integrity | 0 |
| 2 | **EMBODIMENT** | Mission engine В· companion voice В· tactical awareness | 500 |
| 3 | **EXECUTION FRAME** | Swarm command В· shadow execution В· combat protocols В· federation | 2,000 |
| 4 | **MONARCH SHELL** | Kernel shift В· seal matrix В· command fusion В· operator memory | 8,000 |
| 5 | **ZERO-POINT FORM** | Full-system override В· quantum doctrine В· autonomy engine | 25,000 |

XP is awarded by the engine on every mission, swarm tick, federation dispatch, and cosmic query. Evolution gates are one-way. There is no downgrade.

## Seal Matrix

Every sensitive operation passes through a four-level seal. No bypasses exist.

```
OPEN        вҶ’  Public read access

DUAL        вҶ’  Two-party approval required
SOVEREIGN   вҶ’  Operator key + doctrine hash (Zero-Point Form only)
```

## Running Locally

```bash
# Install Python dependencies
pip install -r requirements-dev.txt

# Start primary server  (port 8000)
python nexusmon_server.py

# Start runtime kernel server
python -m swarmz_runtime.api.server

# Start cockpit  (port 5173)
cd cockpit && npm install && node node_modules/vite/dist/node/cli.js

# Or use start.bat (Windows вҖ” launches both servers + browser)
start.bat

# Run tests
pytest tests/ --tb=short -q
```
